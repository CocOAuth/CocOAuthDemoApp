# CocOAuth Demo App

##Installation Steps

1. Clone the App Project
```
$ cd <TARGET_DIR>
$ git clone https://github.com/CocOAuth/CocOAuthDemoApp.git
```

2. Install CocoaPods
https://cocoapods.org/

3. Install the Pod-Project and create a workspace
```
$ pod update
```

4. Open App Workspace 
$ open CocOAuthDemoApp.xcworkspace
